# ☕ Lumière Coffee - Premium Landing Page

Landing page estetik untuk bisnis kopi modern. Dibangun dengan fokus pada **performa**, **SEO**, dan **pengalaman pengguna** yang mulus.

### 🚀 Tech Stack
* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **Animation:** Framer Motion
* **Iconography:** Lucide React / React Icons

### ✨ Features
* Responsive Design (Mobile First)
* Glassmorphism Navigation
* Infinite Testimonial Marquee
* Smooth Scroll & Side Drawer Menu
